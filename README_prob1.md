## Hyperparameter Configurations

Three configurations are trained for both CBOW and Skip-gram:

| Config | Embedding Dim | Window Size | Negative Samples | Notes |
|--------|:---:|:---:|:---:|-------|
| **A** | 50 | 3 | 5 | Fast; lower capacity |
| **B** | 100 | 5 | 10 | ✅ **Best overall** — selected as final model |
| **C** | 300 | 9 | 20 | Highest capacity; slowest to train |

**Why Config B is best:** It achieves the highest composite score across four evaluation metrics — intra-cluster semantic coherence, analogy Mean Reciprocal Rank, average top-1 neighbor similarity, and probe word vocabulary coverage — striking the optimal balance between embedding expressiveness and corpus size constraints (~82K tokens).

To change the number of training epochs, modify `EPOCHS` in Cell 11 (default: `30`). Reduce to `5` for a quick test run.

## Troubleshooting

**`FileNotFoundError: Expected 'corpus_raw.txt'`**  
→ You uploaded a file with a different name. Rename your file to exactly `corpus_raw.txt` and re-run Cell 6.

**`KeyError: 'research' not in vocabulary`**  
→ Your corpus is too small or the word is absent. Ensure `corpus_raw.txt` contains substantial IIT Jodhpur text (at least ~500 KB recommended).

**Cell 11 is extremely slow**  
→ Reduce `EPOCHS` from `30` to `5` in Cell 11 for a quick test. Full training at 30 epochs takes 15–60 minutes depending on corpus size.

**`ModuleNotFoundError: No module named 'google.colab'`**  
→ You are running locally, not on Colab. Replace Cell 6 with:
```python
with open("corpus_raw.txt", "r", encoding="utf-8") as f:
    raw_text = f.read()
raw_documents = [raw_text]
print(f"Loaded {len(raw_text):,} characters.")
```
Then place `corpus_raw.txt` in the same directory as the notebook and run with `jupyter notebook`.

**Word cloud or plots not displaying**  
→ Run `%matplotlib inline` in a new cell before Cell 9, then re-run the plot cells.

**Session disconnects during Cell 11 training**  
→ Colab free tier disconnects after ~90 minutes of inactivity. Keep the browser tab active, or use Colab Pro for longer runtimes. You will need to re-run all cells from the beginning after a disconnect.
